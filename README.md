# MultiWall

Multi-chain wallet dashboard for Solana and Ethereum.

## Setup

```bash
npm install
npm run dev
```

## Features

- Connect Phantom wallet for Solana
- Connect MetaMask for Ethereum
- View real-time SOL and ETH balances
- Responsive layout

## Stack

- React + TypeScript + Vite
- Solana Wallet Adapter
- wagmi + viem
- Tailwind CSS
